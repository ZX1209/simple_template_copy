## 使用
### 确定模板文件夹
默认位置在
~/tempy_templates_dir

可以通过环境变量来配置
tempy_templates_dir

### 获取帮助?
tempy 
tempy -h
tempy --help

### 搜索模板
tempy {template_str}

### 应用复制模板
tempy {template_str} {target_str}

可以有多个 target_str
